# CS132-Compiler-Construction

### Course objective: understand compilers and build one yourself!

### Description
The compiler is broke down into five parts, and thus five total homeworks.


|Homework | Task |
|-------|--------------|
| HW1  | Program Parser (Lexical Scanner) |
| HW2 | Type Checker  |
| HW3     | Logic Checker |
| HW4    | Vapor to VaporM |
| HW5    | VaporM to MIPS  |
